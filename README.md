# 🖋️ Dream Ink Tattoo Studio - Portfolio & Booking System

Sistema dinâmico desenvolvido para um estúdio de tatuagem em Tondela, Portugal. O projeto foca-se na automação de conteúdos e na gestão eficiente de pedidos de agendamento.

Caso queira aceder o link do site é: https://dreaminktattoo.com/

## 🚀 Funcionalidades Principais

* **Galeria Dinâmica:** Integração com a API do Instagram para exibição automática dos trabalhos mais recentes do artista.
* **Gestão de Agendamentos:** Sistema de contacto robusto utilizando EmailJS para envio de formulários diretamente para o email do estúdio.
* **Automação de Dados:** Sincronização de informações do backend (como anos de experiência) inseridas dinamicamente no frontend.
* **Interface Premium:** Design dark mode com acabamentos em dourado, focado na experiência do utilizador e exclusividade da marca.

## 🛠️ Stack Tecnológica

* **Backend:** Python com **FastAPI** (Processamento de APIs e lógica de negócio).
* **Frontend:** HTML5, Tailwind CSS, Alpine.js e JavaScript Assíncrono.
* **Integrações:** EmailJS (Service & Template) e Instagram Graph API.

## 🧠 Áreas de Foco em Engenharia de Software

### 🔗 Integração de APIs
Implementação de fluxos de dados entre múltiplos serviços externos e o sistema local, garantindo a integridade da informação.

### 🤖 Automação
Redução de manutenção manual através da atualização dinâmica de componentes da UI baseada em respostas do servidor.

### ⚡ Operações Assíncronas
Utilização de `fetch` e promessas em JavaScript para garantir que as requisições ao backend e serviços cloud não bloqueiem a performance da página.

### 🛡️ Segurança
Abstração de lógica sensível no servidor e proteção de fluxos de integração para garantir a robustez da aplicação.
