This website was made to a tatto shop at Tondela, Portugal.
On the backend i've implemented functions to get post from instagram account of the ower of the store, funtion to improve security and to automatizate some elements at the frontend.
In the frontend we have some javascript funciton on the bootom of the file index.html, this funcirons were made to end email using emialJS, and to get info from backend, so the backend get the info and de javascript take that info and insert it in the correct space.

In this projects the most important thing i've work with was:
    -> API Integration (Instagram Graph API & EmailJS)
    -> Automation (Dynamic data updates from backend to frontend)
    -> Security 
    -> Asynchronous Operations (Non-blocking fetch requests and form submissions)
